
 Smart Irrigation AI

 Overview
This project is a machine learning and deep learning solution designed to predict irrigation needs (Low, Medium, High) using environmental and soil data. It aims to support smarter and more efficient water usage in agriculture.


 Objective
To build an intelligent model that helps farmers and agricultural systems make data-driven irrigation decisions based on climate and soil conditions.



 Dataset Features
The model is trained on environmental and soil-related features such as:
- Temperature  
- Humidity  
- Rainfall  
- Soil properties  



 Models Used
- Neural Network (Deep Learning)  
- XGBoost Classifier (Machine Learning)  



 Workflow
1. Data preprocessing  
2. Feature engineering  
3. Exploratory Data Analysis (EDA)  
4. Model training (ML & DL)  
5. Model evaluation  



 Performance
The final model achieved an accuracy of **96.3%**, demonstrating strong predictive performance.



 Key Insight
The project shows how combining machine learning and deep learning can improve decision-making in agricultural systems.

 Technologies
